### Setup & Usage

#### Node.js Version Management

This project uses Node.js version 22 (LTS). To ensure you're using the correct version:

**With nvm:**

```bash
nvm use
```

**With fnm:**

```bash
fnm use
```

**With nodenv:**
The correct version will be automatically selected when you enter the project directory.

#### Build Commands

You can use Make with the following commands:

```bash
make  install    - Install dependencies and set up git hooks
make  clean      - Clean node_modules and package-lock.json
make  uninstall  - Remove git hooks and clean dependencies
make  help       - Show this help message
```
